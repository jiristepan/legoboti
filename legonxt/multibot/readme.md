Stavba robota
=============

Robot je postaven podle:

http://www.nxtprograms.com/NXT2/multi-bot/vehicle/steps.html

Zapojení:
* IN 1: senzor light
* IN 2: senzor ultrasonic
* IN 3: senzor zvuk
* OUT_B: motor levy pas
* OUT_C: motor pravy pas
